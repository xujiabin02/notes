



^[+]{0,1}(\d+)$





> (19|20)\d\d([- /.])(0[1-9]|1[012])\2(0[1-9]|[12][0-9]|3[01])

```excel
1900-01-01 2007/08/13 1900.01.01 1900 01 01 1900-01.01 1900 13 01 1900 02 31
```

|                     |                        |      |
| ------------------- | ---------------------- | ---- |
| 正数、负数、和小数  | ^(\-\|\+)?\d+(\.\d+)?$ |      |
| 正整数              | ^[+]{0,1}(\d+)$        |      |
| 整数(包含0，负整数) | ^\-?([0-9]{1,})$       |      |
| 正整数+小数         | ^(\+)?\d+(\.\d+)?$     |      |





```yml
正数、负数、和小数: ^(\-|\+)?\d+(\.\d+)?$

正整数+小数: ^(\+)?\d+(\.\d+)?$
```
